description: Cria a fundação perfeita para um novo componente React, incluindo CSS isolado.
---
Para criar um componente React com excelência estética em minutos:

1. Pergunte à Gabriela o nome do componente e a pasta de destino (caso não esteja muito óbvio).
2. Crie o arquivo fundamental em `<PastaDestino>/<NomeDoComponente>.tsx`.
3. Crie o arquivo visual `<PastaDestino>/<NomeDoComponente>.css` ou integre com o Tailwind, se esse for o direcionamento global dela.
4. No arquivo `.tsx`, traga componentes de base, faça a importação do estilo, e traga as cores e estética visual (sem deixar componentes desanimados/sem graça). Implemente dark mode e modernidade.
5. Deixe pronto e exiba o resultado para avaliação visual.
